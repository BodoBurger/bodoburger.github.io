System Path

  • Anaconda-FAQ on whether you should add Anaconda to the PATH

Temporarily

To add Anaconda temporarily to the PATH, open a terminal and type:

export PATH="/home/bodo/anaconda3/bin:$PATH"

Show PATH:

echo $PATH
